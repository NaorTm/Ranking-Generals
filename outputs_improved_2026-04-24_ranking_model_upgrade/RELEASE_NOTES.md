## Release Framing

This release should be read as a conservative, audited ranking framework with confidence limits.

It should not be read as a final exact-order historical truth for every adjacent commander rank.

Primary reading order:

1. trust tier
2. confidence label
3. exact rank

## What This Release Claims

- the full ranked commander population behind the published outputs was reviewed
- the system is now globally defensible at the framework level
- the ranking is auditable from frozen snapshot to verification outputs
- residual ambiguity is visible and documented

## What This Release Does Not Claim

- that every adjacent ordering is fully settled
- that weak retained-page evidence has been force-interpreted into certainty
- that unresolved commanders imply pipeline breakage

## Why Unresolved Cases Still Exist

Unresolved cases remain because the method is conservative:

- retained pages sometimes contain ambiguous result text
- some pages remain structurally broad even after verification hardening
- the system refuses to over-interpret weak retained-page evidence

The remaining weakness is visible ambiguity, not silent breakage.
